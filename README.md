# Shardfall Depths

Shardfall Depths is a compact browser-based 2D exploration platformer inspired by the feel of classic sci-fi search-action games. It runs entirely on a canvas with original shapes, synth-driven sound effects, fullscreen support, controller input, room-by-room progression locks, and a vault boss encounter.

## Controls

- Move: `A` / `D` or arrow keys
- Jump: `Z` or `Space`
- Beam shot: `X`
- Missile: `V`
- Aim: `W` / `S`
- Dash: `C` or `Shift`
- Sync at stations: `E`
- Fullscreen: `F`

Controller mappings:

- Move: left stick or d-pad
- Jump: `A`
- Beam: `X` or right trigger
- Missile: `B` or left trigger
- Dash: shoulder buttons
- Interact: `Y`
- Fullscreen: `Start`

## Objective

Clear each room to release bulkheads, recover the Aerial Rig and missile tank, defeat the Vault Warden for the final shard, then unlock the eastern extraction gate and escape.

## Scripts

- `npm run dev` starts the Vite development server
- `npm run build` compiles TypeScript and creates a production build
- `npm run preview` serves the production build locally
